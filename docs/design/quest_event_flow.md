# イベントフローチェックリスト

## Q001: 薬草採集クエスト

### 開始フロー
```
dialog: 「町に入りました」
↓
プレイヤー: 村人(1,1)を踏む
↓
dialog: 「おお、冒険者さん！薬草を1つ取ってきてほしいんだ」
↓
クエスト受注: Q001開始
```

### 進行フロー
```
プレイヤー: 町から薬草エリアへ移動
↓
dialog: 「薬草エリアに到着した」
↓
プレイヤー: 薬草(2,0)を踏む
↓
dialog: 「薬草を見つけた！村人に届けよう」
↓
インベントリ: herb +1
```

### 完了フロー
```
プレイヤー: 町に戻る
↓
プレイヤー: 村人(1,1)を踏む
↓
dialog: 「薬草を持ってきてくれたか！報酬だ」
↓
報酬: 100G
↓
クエスト完了: Q001終了
```

## Q002: 魔物討伐クエスト

### 開始フロー
```
プレイヤー: 村長の家(2,0)を踏む
↓
dialog: 「最近、ダンジョンに魔物が住み着いてな...」
↓
dialog: 「討伐の証として、魔物の角を持ってきてくれ」
↓
クエスト受注: Q002開始
```

### ダンジョン探索
```
プレイヤー: ワールドマップ→ダンジョン入口へ
↓
dialog: 「ダンジョン1階に到着」
↓
プレイヤー: 下り階段(2,0)を踏む
↓
マップ遷移: dungeon_01_f2へ
↓
dialog: 「ダンジョン2階に降りた」
```

### 戦闘フロー
```
プレイヤー: 魔物(2,0)を踏む
↓
dialog: 「魔物が現れた！」
↓
戦闘開始: ダンジョンモンスター
↓
戦闘勝利
↓
dialog: 「魔物を倒した！魔物の角を手に入れた」
↓
インベントリ: monster_horn +1
```

### 完了フロー
```
プレイヤー: 町に戻る
↓
プレイヤー: 村長の家(2,0)を踏む
↓
dialog: 「魔物の角を持ってきたか！報酬を受け取ってくれ」
↓
報酬: 500G
↓
クエスト完了: Q002終了
```
